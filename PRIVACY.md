# 隐私政策

**最后更新：2025 年 3 月**

感谢你使用 AI 智能标签分类（以下简称"本扩展"）。本隐私政策说明本扩展如何处理你的数据。

---

## 1. 数据收集

本扩展**不会**向扩展开发者或任何第三方服务器发送数据。所有数据仅存储在用户本地。

### 本扩展收集和处理的本地数据

| 数据类型 | 用途 | 存储位置 |
|---------|------|---------|
| 网页 URL | 作为标签记录的键值，用于在历史记录中检索和展示 | `chrome.storage.local`（本地） |
| 网页标题 | 展示在弹窗和历史记录中，便于识别 | `chrome.storage.local`（本地） |
| 网页正文内容 | 发送到你配置的 AI 接口（如 DeepSeek / OpenAI）用于生成标签，生成后立即释放，不持久保存 | 仅暂存于内存，不写入存储 |
| AI 生成的标签 | 按 URL 保存，供后续查看和管理 | `chrome.storage.local`（本地） |
| 收藏夹数据 | 读取书签列表供批量处理功能使用，不修改或上传 | 仅在运行时读取，不持久保存 |
| API Key | 加密后保存在本地，用于调用你指定的 AI 接口 | `chrome.storage.local`（本地，XOR + Base64 混淆） |
| 主题偏好 | 浅色/深色/跟随系统 | `chrome.storage.local`（本地） |

### 本扩展不收集

- 个人身份信息（姓名、邮箱、电话等）
- 浏览历史（除书签外）
- 登录凭证或密码
- 页面中的表单数据
- 任何分析或追踪数据

---

## 2. 数据共享

本扩展**不向扩展开发者或任何第三方传输数据**。

唯一的网络请求是发往你**自己配置**的 AI API 接口地址（如 `https://api.deepseek.com/chat/completions`），请求内容为：

- 网页正文（用于生成标签）
- 你配置的 API Key（用于接口鉴权）

这些请求受你选择的 API 服务商的隐私政策约束。本扩展开发者无法访问这些请求的内容。

---

## 3. 数据存储与安全

- 所有数据使用 Chrome 浏览器内置的 `chrome.storage.local` 沙盒存储，其他扩展和网站无法访问
- API Key 在存储前经过 XOR + Base64 混淆处理
- 卸载扩展时，所有本地存储的数据将被 Chrome 自动清除

---

## 4. 权限说明

本扩展申请以下 Chrome 权限并说明其用途：

| 权限 | 用途 |
|------|------|
| `storage` | 本地保存标签记录和设置 |
| `activeTab` | 读取当前活动标签页的网页内容以生成标签 |
| `scripting` | 向网页注入内容提取脚本 |
| `bookmarks` | 读取收藏夹以支持批量处理功能 |
| `http://*/` / `https://*/` | 批量处理时打开书签 URL 并提取内容 |

---

## 5. 用户控制

- 你可以随时在设置页修改或删除 API Key
- 你可以通过弹窗或设置页修改、删除任意标签记录
- 你可以随时在 Chrome 扩展管理页禁用或卸载本扩展
- 卸载后所有本地数据将被清除

---

## 6. 政策更新

本隐私政策可能随功能更新而调整。重大变更时会在扩展内通知。
